Pendrive Bootavel (CMD e Diskpart)

O que é um Pendrive bootavel?

Os pendrives bootáveis são unidades de armazenamento portáteis, como pen drives ou discos externos, que contêm um sistema operacional inicializável. Eles são úteis em situações onde você precisa executar um sistema operacional em um computador sem um disco rígido funcional ou quando deseja testar um sistema operacional sem instalá-lo permanentemente.

Para criar um pendrive bootável, você geralmente precisa de um arquivo de imagem ISO do sistema operacional que deseja instalar ou executar, e um software de criação de pendrive bootável, como o Rufos, Ventoy ou o utilitário de criação de mídia de instalação do Windows. Você conecta o pendrive ao computador, seleciona o arquivo de imagem ISO e segue as instruções do software para criar o pendrive bootável.

Uma vez criado, você pode conectar o pendrive a qualquer computador compatível, reiniciar o sistema e iniciar a partir do pendrive. Isso permite que você execute o sistema operacional contido no pendrive sem modificar o sistema operacional principal do computador.

Os pendrives bootáveis são frequentemente usados para propósitos de recuperação de dados, instalação de sistemas operacionais alternativos, como o Linux, ou para fins de diagnóstico de hardware. Eles são uma ferramenta valiosa para técnicos de informática, administradores de sistemas e usuários avançados que precisam de flexibilidade na manutenção e solução de problemas de computadores.

Neste tutorial será aborado o metódo de criação do boot usado o CMD, em linha de comnado.

Procedimentos:

1º) Digite na barra de pesquisa do menu iniciar o comando CMD, depois clique com o botão direito sobre o icone em seguida clique em Executar como Administrador.

2º) Então você vai visualizar a tela preta do Prompt de Comando o CMD.

3º) Conecte o pen drive bootavel em uma das suas saídas USB e certifique-se que ele é o único pendrive conectado em seu PC, e certifique-se que você sabe qual é a capacidade total dele, ex. 4GB ou 8Gb.

4º) Digite o seguintes comandos na sequência:

  • Digite: diskpart
  • Depois: disk
  • Depois: disk list
    • Irá aparecer uma lista de discos de aramzenamento, um deles é o seu pendrive, deve identificar pelo tamanho.
      Cuidado para não fazer o procedimento no disco errado.
  • Após identificar o pendrive digite os seguintes comandos:
    • select disk X (substitua o x pela letra/numero do seu pendrive que aparece na lista)
    • clean (comando para limpar as partições do pendrive)
    • create partition primary (comando para cirar uma partição primaria)
    • select partition 1 (comando para selecionar a partição criada)
    • active (comando para definir a partição como ativa)
    • format fs=fat32 (comando para formatar o pendrive, caso for usar uma ISO maior que 4GB use NTFS no lugar de fat32)
    • assign(comando que monta o pendrive)

5º) Copie todos os arquivos de dentro do DVD de Instalação ou da imagem iso do Windows que queira usar para dentro do pen drive.

6º) O pendrive esta pronto para ser usado!
